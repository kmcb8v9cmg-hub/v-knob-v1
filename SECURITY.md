# Security and privacy

Volume Knob is intentionally local-only.

- It makes no network requests.
- It does not include analytics or crash-reporting services.
- It does not request microphone, camera, screen-recording, contacts, or location access.
- It does not capture or analyze audio. The green pulse is visual only.
- It changes only the current macOS output volume and mute state.

To report a security issue, open a private security advisory in the repository rather than a public issue.

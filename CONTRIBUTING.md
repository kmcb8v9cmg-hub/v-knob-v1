# Contributing

Contributions are welcome.

1. Fork the repository and create a focused branch.
2. Build with `./scripts/build-app.sh`.
3. Test knob dragging, slider changes, volume buttons, mute, Bluetooth output, and menu-bar behavior.
4. Open a pull request describing the change and the macOS version tested.

Please preserve the local-only privacy model. Features that record audio, transmit telemetry, or require unnecessary system permissions should not be added by default.
